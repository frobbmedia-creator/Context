# Context Logo Pack

Approved direction: nested context/conversation frames resolving into a distinctive **C**, with a cyan information accent.

## Files

- `context-logo-horizontal.png` — primary website and document lockup
- `context-symbol-master.png` — standalone symbol master
- `context-logo-stacked.png` — compact vertical lockup
- `context-logo-dark.png` — reversed treatment for dark surfaces
- `context-logo-monochrome.png` — one-color black treatment
- `context-app-icon-512.png` — app store and high-resolution profile icon
- `context-app-icon-192.png` — PWA and social profile icon
- `context-favicon-64.png` — large favicon
- `context-favicon-32.png` — standard favicon
- `context-logo-system-sheet.png` — approved system overview

## Brand colors

- Context Indigo: `#635BFF`
- Signal Cyan: `#22D3EE`
- Context Charcoal: `#111827`
- White: `#FFFFFF`

## Usage

- Use the horizontal lockup when sufficient width is available.
- Use the standalone symbol for app icons, avatars, favicons, and compact navigation.
- Use the dark treatment on charcoal or similarly dark backgrounds.
- Use the monochrome treatment when color reproduction is unavailable.
- Maintain clear space around the mark equal to the width of the cyan accent.
- Do not recolor individual frame layers, distort proportions, rotate, add shadows, or place the mark on visually noisy imagery.

## Source

The approved raster master was generated with OpenAI’s built-in image-generation tool using the `logo-brand` workflow. These exports are raster assets; create a professionally traced SVG master before large-format print production.
